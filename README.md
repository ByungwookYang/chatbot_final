# 프로젝트 소개
- 챗봇 사용 링크 : https://chatbot-for-survey.streamlit.app/


## 설치

다음의 명령어로 가상환경을 활성화 합니다.

```bash
poetry shell
```

패키지를 설치합니다.

```bash
poetry install
```

## 실행

```bash
poetry run streamlit run main.py
```

## 패키지 추가
### 1. poetry.toml 파일에 해당 패키지 이름과 버전을 입력
### 2. 
```bash
poetry lock
```

### 3.
```bash
poetry shell
```

### 4.
```bash

```
